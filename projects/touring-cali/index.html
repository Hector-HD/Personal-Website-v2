Touring California - Microservice

Summary

Touring California - Microservice March 2021 - Current
Tempe, AZ

    Touring California is a microservice that is built on Spring for backend API calls for tour packages and customer ratings. I developed this microservice to understand and continue to learn how to utilize Spring to develop API's. By using Spring I was able to remove a lot of boiler plate code that keeps repeating. Within this project I used JPA to hava data persistence in the application. To make sure the API endpoints were being called I added logging functionality to the service. After the API's were working I created JUnit test cases for the API's.
