UpperBound August 2017 - December 2017
Intro to Software Engineering Project Tempe, AZ

    During the Fall of 2017, we were required to create an Intelligent Tutoring System to help students learn a college subject for our CSE360: Intro to Software Engineering class. Each group had four members and were given the tasks to implement Software Engineering design patters, an Avatar to interact with, Graphical User Interfaces(GUI), and create Use Cases for the program. My team and I decided to teach students calculus. Each one of us were given the task to complete a certain part of the project. Once we had our parts done, we came together to put the program together. There were five phases throughout the semester. Each phase would be a continuation of the previous phase. This project helped me out with realizing how much planning, documentation, and team work building working software takes.
